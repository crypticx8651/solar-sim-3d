# solar-sim-3d


 # 🌌 Solar System Simulation

An interactive 3D solar system simulation with realistic orbital mechanics, time control, and dynamic camera navigation.

## 🚀 Features
- Realistic planetary orbits (Kepler-based)
- Time scaling (speed up / slow down simulation)
- Jump to custom dates
- Focus on planets and celestial bodies
- Toggle orbits, labels, moons, and dwarf planets
- Smooth camera controls (pan, zoom, focus)

## 🧠 Tech Used
- HTML
- CSS
- JavaScript
- (Add Three.js here if you used it)

## 🎮 Controls
- Drag → Pan camera  
- Scroll → Zoom  
- Click object → View details  

## 📌 Notes
This is a high-fidelity visualization using Keplerian elements, not a full n-body physics simulation.
